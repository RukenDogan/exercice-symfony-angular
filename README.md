# Enoncé :

* 0° Créer un projet Symfony et y ajouter Angular.
* 1° Créer une base de données d’utilisateurs comportant les informations inclus dans le fichier "data_BDD_user.jpg" en lien ci-dessous. *(vous pouvez utiliser votre projet Symfony issue de la formation Symfony)*

* 2° Afficher les informations dans un tableau. La liste des utilisateurs
  doit impérativement se trouver dans, au moins, un composant ReactJS.** (Il est interdit d'utiliser API_Plateform).**
* 3° ajouter un bouton supprimer(). Supprime un utilisateur du tableau.

* 4° Ajouter une autre table contenant les possessions de la personne comportant les informations inclus dans le fichier "data_BDD_possession.jpg" en lien.
* 5° Afficher ces informations sur une autre page quand on clique sur le nom de la personne. On doit retrouver les informations de la personne ainsi que ses possessions dans un tableau. Le tout doit se trouver dans un composant Angular.

* 6° Ajoutez un champ birthDate dans la table utilisateur comportant l'information inclus dans le fichier "data_BDD_user_addBirthDate.jpg".
* 7° Créer un service(UserService) qui permet de calculer son âge. Et ensuite afficher son âge dans le premier tableau affichant tous les utilisateurs. La transmission des informations doit se faire via une API et issue de l'entité "User" *(on doit retrouver l'information de l'âge dans l'entité user sans pour autant qu'elle soit enregistrée dans la BDD).*

* 8° Créer une fenêtre modal sur la page de la liste des utilisateurs permettant d'ajouter un nouvel utilisateur*(seul les informations de l'utilisateur doit s'y trouver, il ne faut pas ajouter les informations des possessions).*
